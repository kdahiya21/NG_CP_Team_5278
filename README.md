StyleMe is a web-based prototype that uses camera input or uploaded
images to generate personalized outfit recommendations based on user
preferences like occasion, style, gender fit, and budget.

Features

-   Capture your look using webcam
-   Upload an image instead of using camera
-   Select Occasion, Fit, Style, and Budget
-   AI-based outfit recommendations (Top, Bottom, Footwear, Colors,
    Tips)
-   Virtual fitting experience
-   AR Try-On prototype (overlay demo)
-   Trending fashion slideshow
-   Modern responsive UI

Tech Stack

Frontend: - HTML5, CSS3, JavaScript

Backend: - Python (Flask)

Browser APIs: - getUserMedia (Camera Access) - Canvas API (Image
Capture) - Fetch API (Backend Communication)

How It Works

1.  User captures or uploads an image.
2.  User selects styling preferences.
3.  Frontend sends data to backend.
4.  Backend generates suggestions.
5.  UI displays personalized outfit recommendations.

Running the Project

1.  Clone Repository: git clone cd styleme

2.  Install Dependencies: pip install requirements.txt

3.  Run Backend: python app.py

4.  Open index.html in browser.

Privacy Note

Captured photos may be saved locally in the saved_photos folder. No
cloud storage is used in this prototype.
